### Comando de Compilacion.

-> Es importante abordar una vez abieto el proyecto en Android Studio, el comando para su compilacion en la terminal integrada de Android Studio. Asi el comando que se debe ejecutar posteriormente de la apertura y la sincronizacion de Gradle:

> ./gradlew android:installDebug

No debe olvidarse que hay dos formas de ejecutar la aplicacion. Esto dependera de si se desea usar un celular fisico  el emulador de android el cual debera considerarse or el uso de recursos. Si se opta por instalar en dispositivo fisico, deberan hacerce previamente las instalaciones necesarias del controlador de Android ADB para que se areconocible en el sistema.

Se deja informacion acerca de esto para que sea posible probar de forma nativa el juego:
